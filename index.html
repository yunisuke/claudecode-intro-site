実際にClaudeCodeでやったこと

最近ClaudeCodeを使って色々試してみたので、その実例を紹介します！

1. モデルごとの性能評価 🔬

Opus4とSonnet4を使ってランディングページを作成し、各モデルの特徴を比較しました。

  • Opus4: より高度な推論が可能で、複雑なタスクに強い
  • Sonnet4: 高速でバランスが良く、日常的な開発タスクに最適

結果: どちらも短時間で高品質なLPを生成できました！

2. UnityMCPと連携させてUnity開発 🎮

ClaudeCodeとUnityMCPを連携させることで、Unity開発が爆速に！

  • シーンの自動生成
  • スクリプトの作成・編集
  • アセットの管理

Unity開発の効率が大幅にアップしました！

3. このページを作った！ 🌐

まさに今見ているこのページもClaudeCodeで作成しました。

  • GitHubリポジトリの作成（gh CLIを使用）
  • HTML/CSS/JavaScriptでのサイト制作
  • GitHub Pagesでの公開設定

わずか30分程度で公開まで完了！
